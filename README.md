# Arduino Uno and W25N01GV  Testing

**Connection:**

| Arduino Uno | W25N01GV |
| ----------- | -------- |
| 10 (SS)     | CS       |
| 11 (MOSI)   | DI       |
| 12 (MISO)   | DO       |
| 13 (CLK)    | CLK      |
| VCC         | WP       |
| VCC         | HOLD     |
| VCC         | VCC      |
| GND         | GND      |
